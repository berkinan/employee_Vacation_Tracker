Kafein Employee Vacation Tracker
Project Overview
Welcome to the Kafein Employee Vacation Tracker! This project represents a substantial step forward in our commitment to improving operational efficiency within Kafein Technology Solutions. Built leveraging modern web technologies, this full-stack web application simplifies the process of tracking and managing our team's vacation days, offering a seamless and intuitive user experience.

Key Features
Employee Registration: Efficiently add new team members into the system using just their email address through a user-friendly interface.
Employee List: Gain a detailed overview of all employees in the system along with the specifics of their vacation days.
Editable Employee Data: Easily modify an existing team member's vacation day count when necessary.
Data Reset Capability: In the event of system-wide updates or overhauls, conveniently reset all stored data.
Getting Started
To successfully set up this project on your local system, ensure that you have the following prerequisites installed:

Node.js
MySQL

Follow the outlined steps to run this application:

-Clone the project repository on your local system.
-Navigate to the project's root directory and install the required NPM packages.
-Create a .env file in the root directory of your project, and include your MySQL credentials.
-Initiate the server.
-In a new terminal window, start the React application.

Technologies Used

The Kafein Employee Vacation Tracker is built using:

Frontend: React.js
Backend: Node.js, Express.js
Database: MySQL
